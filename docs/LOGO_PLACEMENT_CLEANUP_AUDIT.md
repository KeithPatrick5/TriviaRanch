# Logo Placement Cleanup Audit

## Goal

Use the selected fourth logo direction without making the home screen feel like a duplicated promo banner.

## Changes

- Removed the full boxed header-logo treatment from the home screen.
- Added a compact transparent lockup for in-app header use.
- Kept the larger plaque-style identity for app/icon/splash-style usage only.
- Removed repeated slogan usage in the home header.
- Kept the rank badge in the header, but reduced its footprint.
- Kept generated screenshot folders out of the repo package.

## Result

Pass. The logo now behaves like part of the interface instead of a content card.
