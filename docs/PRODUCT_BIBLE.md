# Neon Trivia Product Bible

Neon Trivia is an Android-first offline trivia game with a late-night quiz lounge style.

## Core identity

Fast trivia rounds. No ads. No filler. One more round.

The app should feel like a neon party quiz show you can play anywhere: couch, bus, cabin, airplane, or dead-zone bathroom.

## Product rules

- Android first.
- Offline first.
- No forced ads.
- No childish trivia-cartoon look.
- No builder-facing copy in the user interface.
- Gameplay must stay fast even when the backend is down.
- Backend is used for official sessions, reports, score sync, and future challenge validation.

## Visual direction

Late-night quiz lounge:

- deep indigo and purple base
- neon pink primary action
- cyan highlights
- violet panels
- dark lounge/game-show mood
- energetic but readable

## V1 modes

- Daily Blitz
- Survival
- Challenge Run
- Pass the Phone

## Question direction

Questions ship offline in bundled packs. APIs and datasets are for import/build tooling only, not runtime gameplay.
