# Trivia Ranch Product Bible

Trivia Ranch is an Android-first trivia fight app. It is not a web-first quiz site, not an ad farm, and not a cartoon learning game.

## Product thesis

Most trivia apps feel slow, cheap, bloated, childish, or stuffed with ads. Trivia Ranch should feel like a fast competitive fight: pick a category, prove yourself, lose by one, run it back.

## Core vibe

- Clean dark interface
- Sharp cards, tight spacing, fast flows
- Ranch gold accent, red/green only for game feedback
- No bubbly AI design
- No kid-game mascot energy
- No ad slots in the UI
- No forced signup before first play

## Main loop

Pick mode -> pick category -> answer fast -> win or lose -> see rank/progress -> run it back or challenge someone.

## V1 modes

1. Daily Blitz: 60 seconds, shared daily seed, score chase.
2. Survival: 3 lives, keep answering until dead.
3. Challenge Run: async challenge payload first, deep links later.
4. Pass the Phone: 2 to 8 players on one device.

## Category hierarchy

Main categories stay broad. Niche categories live under them.

- Sports contains NFL, NBA, MLB, F1, UFC, boxing, soccer, and similar.
- Science contains biology, chemistry, physics, space, animals, and similar.
- Math contains mental math, algebra, geometry, probability, logic, and similar.

## Question quality rule

Better to have 5,000 strong questions than 200,000 AI-generated trash questions. Every public-launch question needs a source note, freshness tag, report path, and performance stats.

## Monetization principle

No forced ads. Premium should unlock depth and unlimited play, not make the free version feel hostile.
